About Me

My name is Bryan Enriquez, a self-taught developer. My journey started early 2020, where for the first few months I worked through the "Responsive Web Design" and "JavaScript Algorithms and Data Structures" curriculums on freeCodeCamp. Having thoroughly enjoyed the courses, I decided to pick up a few Udemy courses. Ok, maybe more than a few. Since then, it's been a journey of non-stop coding.

Recently I've been developing my front-end skills through some of the challenges on frontendmentor.io. Aside from my current projects, I'm interested in working on larger real-world projects in the industry. And just like my skill set, this website is constantly growing. Expect to see more projects soon!

Product Feedback App

Description: Here's my solution to yet another frontendmentor.io challenge. It's a simple CRUD app where users can leave feedback for a product and others can share their thoughts by leaving a comment. Other features include sorting suggestions by upvotes/comments, filtering by category, and upvoting other suggestions.

Feel free to log in as a demo user and leave a random suggestion or comment!

Designo Multi-page Website

Description: The project idea is from frontendmentor.io but the solution is my own. The challenge was to build out a 7-page website using any technology. To keep things simple, I went with the basic create-react-app setup. While I could have used Next.js I wanted to familiarize myself with the latest version of react-router.

Since it's just a static website, most features were easy to implement. The only challenging part was the CSS styling as usual. It's the bane of my existence. I'd have definitely taken much longer if I wasn't using a preprocessor like SASS and the BEM naming convention.

Built with:

  • React
  • React router
  • Mapbox
  • SCSS
