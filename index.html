My Projects

Audiophile - eCommerce Site

"You just can't have a portfolio site without an e-Commerce project!" - Someone out there, probably.

For this project I took the opportunity to learn a couple new things: Nextjs, TypeScript, and how to work with a headless CMS. Development was pretty straightforward given that it's mostly all static content. Pages are pre-rendered ahead of time, app state is managed using React Context API, and data is persisted on page refresh using localStorage.

Built with:
  • Nextjs
  • TypeScript
  • Strapi
  • SCSS Modules
  • Amazon S3

Product Feedback App

Here's a simple full-stack app built with the PERN stack. Users can leave feedback for a product, in this case what improvements can be made to the site. Other users then have the ability to share their thoughts on the subject as well. Other features include sorting suggestions by upvotes/comments, filtering by category, and upvoting other suggestions.

Feel free to log in as a demo user and leave a random suggestion or comment!

Built with:
  • Node
  • Express
  • PostgreSQL
  • Sequelize
  • React
  • React-Router
  • Redux-Toolkit
  • SCSS
